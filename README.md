# Loja das prioridades

A minigame made for "Grupo de Jovens de S. Miguel-O-Anjo.

## Setup

1. Create a project in [Supabase](https://supabase.com/).
2. Copy the `.env.example` to `.env`
3. Put your API url and API publishable key into the .env
4. Run the app via `npm run dev` or `bun run dev` or deploy it to vercel.
